# 名词解释

## 结构化数据

用户输入的是自然语言：
```
我明天从杭州去上海出差，预算1000元，希望坐高铁
```
这句话人能看懂，但是程序不好直接使用。

所以我们要把它变成字典：
```
{
    "出发地": "杭州",
    "目的地": "上海",
    "出行时间": "明天",
    "预算": "1000元",
    "偏好": ["高铁"],
    "约束": [],
    "原始输入": "我明天从杭州去上海出差，预算1000元，希望坐高铁"
}
```
字典的好处：后面的函数可以直接取值。

未来写行程规划函数时，就可以利用提取出的结构化数据进行取值：
```
departure = trip_info["出发地"]
destination = trip_info["目的地"]
budget = trip_info["预算"]
```
