# 差旅出行助手

这是一个用于持续迭代的差旅出行助手项目仓库。

项目目标是逐步构建一个能够辅助差旅规划、行程整理、费用记录和出行建议的智能助手。仓库会按阶段记录每次升级内容，方便回顾项目从 MVP 到可用产品的演进过程。

## 当前状态

- 项目阶段：MVP 起步
- 主要语言：Python
- 当前入口：`main.py`

## 迭代记录方式

每次完成一个阶段后，建议同步更新：

- `CHANGELOG.md`：记录已经完成的版本变化
- `ROADMAP.md`：记录后续计划和优先级
- Git commit：用简短提交信息描述本次变化

## 推荐提交信息格式

```text
feat: 新增某个功能
fix: 修复某个问题
docs: 更新文档
chore: 项目配置或依赖调整
refactor: 重构但不改变功能
```

## 本地运行

```powershell
.\.venv\Scripts\python.exe main.py
```
