# CHANGELOG

本文件用于记录「差旅出行助手」项目每个版本的主要变更。

项目采用 MVP 可迭代方式开发：

```text
先提出项目问题
↓
再分析需要什么东西
↓
再写最小代码
↓
再逐步迭代成完整项目
```

## [0.3.0] - 2026-06-29

### added
* 定义用户意图类型
* 编写`detect_intent()`用户识别函数
* 输出关键词判断用户需求
* 设置默认意图`chat`
* 输出识别结果

### notes

本版本完成了差旅出行助手的基础意图识别能力。

系统已经可以初步判断用户输入属于行程规划、信息查询、偏好记录或普通聊天，为后续任务调度和多Agent编排打下基础。

---

## [0.2.0] - 2026-06-29

### added

* 支持命令行输入用户需求
* 使用变量保存用户输入
* 输出基础助手回复
* 建立 `main()` 程序入口
* 跑通最小流程：用户输入 → 程序接收 → 程序回复
### notes
本版本完成了差旅出行助手的最小输入输出闭环

虽然系统暂时还不能够理解用户的需求，但是已经具备了AI助手最基础的交互结构。后续版本将在这个基础上加入意图识别能力。

---

## [0.1.0] - 2026-06-29

### Added

* 创建 Python 项目入口

* 创建 `roadmap.md`，用于记录项目迭代路线

* 后续 MVP 开发方向

下一版本将进入命令行最小闭环开发。
